# TicTacToeMCTS
Tic Tac Toe terminal game utilising Monte Carlo Tree Search as AI

The program has 3 game modes:
    1. Human VS Human
    2. Human VS MCTS_AI
    3. MCTS_AI VS MCTS_AI

Run TicTacToe MCTS.exe to play!
